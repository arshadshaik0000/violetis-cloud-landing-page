# Violetis Cloud Landing Page

A modern, responsive landing page built with **Angular** to showcase the features of **Violetis Cloud**, an AI infrastructure platform.  

## Features
- Responsive design (desktop, tablet, mobile)  
- Hero with parallax & animations  
- AI API interaction demo  
- Developer dashboard with metrics  
- Light/Dark theme toggle  

## Tech Stack
Angular 20 · TypeScript · SCSS · Angular Material · Netlify
